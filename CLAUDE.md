# KreativLab - Production Commands

## Development Commands
```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

## Database Commands
```bash
npx prisma migrate dev
npx prisma generate
npx prisma studio
npx prisma db push
```

## Testing Commands
```bash
npm test
npm run test:e2e
npm run test:unit
npm run test:coverage
```

## Production Commands
```bash
npm run build
npm start
npm run migrate:prod
```

## Docker Commands
```bash
docker-compose up -d
docker-compose down
docker-compose logs -f
```

## Security Audits
```bash
npm audit
npm audit fix
```
